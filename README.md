<h1>Student Management System Python</h1>
A lightweight command-line interface (CLI) tool for managing student data. Built with a focus on Object-Oriented Programming (OOP) principles and persistent file storage. Add, Update, View, Search, and Delete student records. Persistent data storage using MySQL. Search functionality by Student ID.

<h2>Features:</h2>
<p>Student Data Management (CRUD operations):The primary function is to manage student records efficiently. This includes:</p>
<h4>1. Adding new students:</h4> <p> Entering details such as name, age, ID, course and email into the MySQL database.</p>
<h4>2. Viewing student details:</h4> <p>Displaying records of all students or a specific student via the command line.</p>
<h4>3. Updating student information:</h4> <p>Modifying existing records (e.g., updating phone numbers, grades, or personal details).</p>
<h4>4. Deleting student records:</h4> <p>Removing student entries from the database, typically using a unique identifier like a student ID or roll number.</p>
<h4>5. Search and Filter Functionality:</h4> <p>Users can search for specific student records using criteria such as ID, name, or roll number to quickly retrieve information.</p>
<h4>6. Database Connectivity:</h4> <p>The system uses a Python library (like mysql-connector-python or pymysql) to establish a connection with the MySQL database, ensuring data integrity and secure storage.</p>
<h4>7. Command Line Interface (CLI):</h4> <p>The system provides a simple, menu-driven command-line interface, allowing users to interact by selecting options. </p>
